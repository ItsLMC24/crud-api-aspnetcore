# ASP.NET Core CRUD API

A RESTful Web API built with ASP.NET Core using clean architecture principles.

## Features
- ASP.NET Core Web API
- RESTful CRUD operations
- SQL Server + Entity Framework Core
- Controller / Service / Repository pattern
- DTO mapping
- Swagger (OpenAPI)

## Entities
- Users
- Products
- Orders
